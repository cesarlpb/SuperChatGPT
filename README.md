# SuperChatGPT

App minimalista para pedir prompts con HTML, CSS y JS :)

Aplicación realizada en grupo como parte de la formación IFCT0609. No guarda información del usuario, funciona solo en lado cliente.

## Tecnologías

- HTML
- CSS
- Javascript
- Openai API
- MySQL - SQL
- PHP :)
